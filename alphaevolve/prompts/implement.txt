== WHAT THE FUNCTION MUST DO ==
1. Walk nodes in topological order using `TopoSort(f)`.
2. Skip any node where `IsUntimed(node)` is true.
3. Read the feasible window `[bounds->lb(node), bounds->ub(node)]`.
4. Choose a cycle inside that window using a principled heuristic
   (timing / register pressure / mobility / lookahead).
5. Record the choice in a `ScheduleCycleMap`
   (`absl::flat_hash_map<Node*, int64_t>`).
6. Pin the choice via `bounds->TightenNodeLb(node, cycle)` and
   `bounds->TightenNodeUb(node, cycle)`.
7. Call `bounds->PropagateBounds()` after each assignment.
8. Return the completed map.
